# RestaurantApp

Evaluación 1 - Programación II

Aplicación Android desarrollada en Kotlin utilizando Android Studio.

## Funcionalidades

- Selección de platos:
  - Pastel de Choclo
  - Cazuela
- Ingreso de cantidades.
- Cálculo automático de subtotales.
- Propina opcional del 10%.
- Cálculo del total final.
- Interfaz gráfica desarrollada con ConstraintLayout.

## Tecnologías utilizadas

- Kotlin
- Android Studio
- XML
- ConstraintLayout
